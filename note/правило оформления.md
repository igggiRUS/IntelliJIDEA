```java
// Мы находимся в пакете Lesson01
package Lesson01;
// Указан публичный класс тест01_принт_принтлн
public class test01_print_println {
    /**
     * @param args the command line arguments
     */
    public static void main(String[] args) {
        // TODO code application logic here Здесь должна быть описанна логика происходящего
        /** 
         * В классе должнен быть метод main это стартовая точка для запуска программы
         * Если в программе нет метода мейн она будет не выполнима
         * Все что находится между фигурмыми скобками будет работать как программа
         * После каждого выражения в java ставим ; это говорит что наш стейтмен завершен
         * print println это метод функции первый выводит на дисплей текст второй переводит строку
         * все что в скобках является параметром функции
         * 
         */
    System.out.print("И по традиции ХЕЛЛОУ Ворлд ");
    System.out.println(); // создает пустую строчку
    System.out.println(); // создает пустую строчку
    System.out.println("И по традиции ХЕЛЛОУ Ворлд");
    System.out.println(); // создает пустую строчку
    System.out.println(); // создает пустую строчку
    System.out.println("Домашнее задание: Вывести на дисплей рубаи (четверостишье),");
    System.out.println("                  соблюдая все отступы следующем виде:");
    System.out.println(); // создает пустую строчку
    System.out.println(); // создает пустую строчку
    System.out.println("                 РУБАИ");
    System.out.println("Много лет размышлял я над жизнью земной.");
    System.out.println("Непонятного нет для меня под луной.");

    System.out.println("                                О. Хайям");
    }
}

```


Такой стиль написания имён называется _[[lowerCamelCase]]_ (с англ. — «[[нижнийВерблюжийРегистр]]»), так как первая буква — обязательно «маленькая», строчная, по-английски — _lowercase letter_, а прописные внутри слова напоминают горбы верблюда.
